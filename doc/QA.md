## Q. 牛さんが邪魔です！

```
 ______________
<   仕様です    >
 --------------
        \   ^__^
         \  (oo)\_______
            (__)\       )\/\
                ||----w |
                ||     ||

```

ansible の ~~うざい~~ 愉快な機能です．デフォルトで有効になっているので，`ansible.cfg` に設定を追加することで無効にできます．

```ansible.cfg
[defaults]
nocows=1
```

余談: この機能のせいで[こんな](https://github.com/ansible/ansible/issues/10530) issue が大量に立てられているとかなんとか．
